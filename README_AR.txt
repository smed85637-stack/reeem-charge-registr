طريقة الاستخدام:

1) لفحص الصفحة:
- افتح ملف index.html في الهاتف أو الكمبيوتر.

2) لنشرها كصفحة ويب مجانية:
- ادخل إلى netlify.com
- اختر Add new site
- اختر Deploy manually
- ارفع ملف reem_charge_web_page.zip أو اسحب ملف index.html
- سيعطيك رابط صفحة ويب جاهز.

3) لإضافتها داخل AppCreator24:
- بعد نشر الصفحة وخروج الرابط
- افتح AppCreator24
- أضف صفحة WebView / Website
- ضع رابط الصفحة
- سمّ الصفحة: سجل الشحن

ملاحظة:
البيانات محفوظة داخل متصفح الجهاز فقط.
إذا أردت أن تظهر البيانات في كل الهواتف، تحتاج نسخة بقاعدة بيانات وتسجيل دخول.
